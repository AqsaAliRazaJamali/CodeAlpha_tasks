# Hangman Game

A simple text-based Hangman game developed in Python as part of my Python Programming Internship at CodeAlpha.

---

## 📌 Project Overview

This project is a console-based implementation of the classic Hangman game. The program randomly selects a word from a predefined list, and the player must guess the word one letter at a time within a limited number of incorrect attempts.

---

## 🚀 Features

- Random word selection using Python's `random` module
- Letter-by-letter guessing
- Maximum of 6 incorrect attempts
- Input validation for single alphabet characters
- Prevention of duplicate guesses
- Displays guessed letters and remaining attempts
- Win and lose conditions

---

## 🛠️ Technologies Used
