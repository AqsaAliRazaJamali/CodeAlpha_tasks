## 📖 Overview

The **Rule-Based Chatbot** is a lightweight Python application designed to simulate basic human-computer interaction through predefined conversational rules.

The chatbot accepts user input from the terminal, standardizes text formatting, and generates responses based on keyword matching using conditional statements. The project demonstrates fundamental chatbot design principles while showcasing essential Python programming concepts.

This project highlights concepts such as **input processing, keyword mapping, control flow, and continuous interaction loops**.

---

## ✨ Features

-  **Keyword-Based Response System** using `if-elif-else` conditions.
-  **Input Sanitization** with `.lower()` and `.strip()` to ensure consistent processing.
-  **Continuous Conversation Loop** allowing multi-turn interactions.
-  **Graceful Exit Mechanism** that safely terminates the chatbot when users enter commands like `"bye"`.
-  **Lightweight Terminal Interface** requiring no external libraries.
